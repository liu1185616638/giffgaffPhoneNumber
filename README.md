之前邮箱注册就能用，现在突然弹出个窗口，让你输入手机号收验证码。+86的号填进去，直接灰掉——不支持。

去搜了一圈，接码平台要么贵得要死，要么码已经被用烂了，收不到。Google Voice呢？风控严的时候一样翻车。

这时候就轮到一个老牌方案出场了：giffgaff。

二、为什么是giffgaff？

先说几个硬指标：

零月租。 选Pay as you go方案，不扣月租，不用不花钱。在国内插卡，只要不开数据漫游，一毛钱都不会被扣。

保号成本极低。 规则是180天内要有余额变动，不然号码回收。最简单的保号方式：每半年发一条短信，成本0.3英镑（约2.7元人民币）。一年两条，五块钱搞定。充一次10英镑，理论上能用十几年。

免实名。 注册填拼音就行，选国家China，不需要英国地址验证。英国预付费卡不强制实名。

真实英国号码（+44）。 不是虚拟号段，在ChatGPT、Codex、WhatsApp、Telegram这些平台的风控系统里是"优质号段"，注册通过率高。

收短信免费。 在国内漫游状态下，接收所有短信零费用。这就是它在国内最大的用途。

说白了，这就是一张长期低成本持有的境外&quot;身份证&quot;。接码平台是租房子，giffgaff是买房子。

四、操作步骤
第一步：注册giffgaff

打开giffgaff官网，用Gmail邮箱注册。这个过程没什么好说的，填邮箱、设密码、验证邮件，两分钟完事。

第二步：让giffgaff认为你的设备支持eSIM

这里是一个小技巧。大部分国行手机硬件不支持eSIM，但我们用模拟器来"骗"一下giffgaff App。

安装一个安卓模拟器（推荐雷电），在模拟器里安装giffgaff App。用LSPosed框架Hook一下eSIM检测接口，这样App就会认为你的设备具备eSIM能力。

详细说明一下这里，关键！！！！！！
1、安装官网雷电9模拟器
2、前往 https://www.duokaiya.com/magisk.html 下载LSPosed框架、Magisk面具
3、上述地址有打包的教程和文件，都安装到模拟器，然后按教程配置。
4、等你的面具安装好了，安装我上传的app-release-sign.apk 这个就是模拟返回有eSIM的模块了
5、打开giffgaff就可以看到不会提示你手机没有eSIM了。


第三步：选择套餐——拉到最下面

登录App后，进入"Choose your plan"页面。别急着选，直接拉到最下面，找到那个不起眼的选项：&quot;I don't want a plan&quot;。

对，就是不要套餐。我们要的是Pay as you go，按量付费，无月租。

第四步：充值10英镑

选择支付方式，输入你的VISA卡信息。招行的双币VISA卡就行，账单地址填拼音。支付10英镑，这笔钱就是你的账户余额，充进去以后慢慢用。

支付完成，eSIM信息就生成了。

这时候去检查你的Gmail邮箱——你的英国手机号已经发过来了。



第五步：下载eSIM信息

回到giffgaff App，找到eSIM下载入口。一路点击确认，最后会自动复制一段激活码到剪贴板。

把这个激活码保存好，接下来要用。



第六步：写入空白SIM卡

现在拿出你的安卓手机和空白SIM卡。

把小白卡插进手机，打开EasyEUICC。如果你的手机支持（大部分安卓机都行），App会提示检测到一张空白卡待写入。

点击右上角的"+"号添加，选择"从剪贴板导入"。刚才复制的激活码会自动填入，确认写入。

等待几十秒，激活完成。



第七步：关掉数据漫游！

这是最关键的一步。

giffgaff的数据漫游费是200英镑/GB，按现在的汇率差不多1800元人民币。忘记关流量，刷几个网页，余额就没了。

所以激活完成后第一件事：进手机设置，把这张eSIM的数据漫游关掉。短信和通话也切到国内主卡，eSIM只用来收码。

第八步：验证Codex

最后一步，也是最爽的一步。

下载WhatsApp，用你的新英国手机号绑定。+44开头那个。

然后去Codex，在手机号验证页面选择区号+44，填入你的giffgaff号码。选择"通过WhatsApp发送验证码"——嘀嗒一声，验证码秒到。

填进去，验证通过。

半年发一条短信保号。 设个日历提醒，每180天内发一条短信（成本0.3英镑），号码就一直在。

虚拟卡可能被风控。 充值建议用实体VISA/MasterCard，有些虚拟卡在giffgaff那边会被拒。
